Введение в CSS

CSS (Cascading Style Sheets) — это язык описания внешнего вида документа. Он позволяет отделить визуальное оформление от структуры HTML, делая код более чистым и удобным для поддержки.

С помощью CSS вы можете задавать цвета, размеры, расположение и другие визуальные характеристики элементов на странице.

Пример базового селектора и свойств:

p { color: #333; font-size: 16px; line-height: 1.5; }

Ключевые преимущества использования CSS:

  • Разделение структуры и оформления (HTML отвечает за семантику, CSS — за дизайн)
  • Повторное использование стилей на разных страницах через один файл
  • Ускоренная загрузка благодаря кешированию внешних CSS-файлов
  • Гибкое управление макетом и адаптивностью (Flexbox, Grid, медиазапросы)
  • CSS поддерживается всеми современными браузерами и постоянно развивается: появляются новые возможности, такие как CSS-переменные, кастомные шрифты и расширенные модули для сеток и анимаций.
Что такое CSS и зачем он нужен

CSS нужен для того, чтобы:

  • Упростить процесс создания и последующего обслуживания веб-сайтов. Вместо того чтобы прописывать правила оформления для каждого отдельного элемента, веб-разработчики могут определить общие стили с помощью CSS и применять их ко многим страницам сайта.
  • Обеспечить единообразие в оформлении связанных документов. Достаточно внести изменения в центральный CSS-файл, и новые стили автоматически распространятся на все подключённые страницы, сохраняя согласованность их внешнего вида.
  • Адаптировать веб-страницы под различные устройства и разрешения экрана. Благодаря использованию медиа-запросов и гибких сеточных раскладок, веб-дизайнеры могут создавать отзывчивые макеты, автоматически подстраиваясь под размеры экрана смартфонов, планшетов, ноутбуков и настольных компьютеров.
  • Создавать анимации. С CSS можно создавать красивые анимации без использования JavaScript
Селекторы

Селекторы в CSS — это шаблоны, которые используются для привязки стилевых свойств к элементам в документе.

Некоторые типы селекторов:

  • Селекторы по тегу. Позволяют выбирать все элементы определённого типа. Например, селектор p выберет все абзацы на странице.
  • Селекторы по классу. Позволяют выбирать элементы, имеющие определённый класс. Класс обозначается точкой перед именем класса.
  • Селекторы по идентификатору. Выбирают элемент с определённым id. Идентификатор обозначается символом решётки # перед именем идентификатора.
  • Универсальный селектор. Выбирает все элементы на странице. Это может быть полезно для сброса стилей по умолчанию или для применения глобальных стилей.
  • Селекторы соседних элементов. Выбирают элементы, которые непосредственно следуют за другим элементом. Это обозначается символом +.
  • Селекторы общих соседей. Выбирают все элементы, которые следуют за другим элементом, но не обязательно непосредственно. Это обозначается символом ~.

Базовые селекторы можно комбинировать между собой, чтобы выбрать элементы на основе нескольких условий.

Цвета и фон

Цвета в CSS можно задавать несколькими способами:

  • С помощью ключевых слов (например, red, blue).
  • С помощью шестнадцатеричных значений (например, #ff0000).
  • С помощью RGB и RGBA значений (например, rgb(255, 0, 0), rgba(255, 0, 0, 0.5)).
  • С помощью HSL и HSLA значений (например, hsl(0, 100%, 50%), hsla(0, 100%, 50%, 0.5)).

Фон в CSS может быть представлен в виде изображений, градиентов или текстур.

Для работы с фонами в CSS есть, например, такие свойства:

  • Background-image. Позволяет задать изображение в качестве фона элемента. Это может быть любой тип изображения, включая jpeg, png, gif, svg и другие.
  • Background-repeat. Определяет, как фоновое изображение будет повторяться внутри элемента, когда оно меньше размера элемента.
  • Background-size. Позволяет установить размеры фонового изображения. Значение может быть указано в пикселях, процентах, которые рассчитываются от размера элемента, или любых других доступных для CSS единицах измерения.
  • Background-attachment. Определяет, будет ли фоновое изображение прокручиваться вместе с содержимым элемента.

Градиенты позволяют создавать плавные переходы между цветами. В CSS существуют линейные и радиальные градиенты. Линейные градиенты создают переходы в одном направлении, тогда как радиальные градиенты создают переходы, исходящие из центра элемента.

Шрифты и текст
  1. Свойства шрифтов. Они влияют на то, какой шрифт применяется, насколько он велик, является ли он полужирным, курсивным и т. д.. Некоторые из таких свойств:
    • font-family. Устанавливает семейство шрифта, которое будет использоваться для оформления текста содержимого.
    • font-style. Определяет начертание шрифта — обычное, курсивное или наклонное.
    • font-weight. Устанавливает насыщенность шрифта.
    • font-size. Определяет размер шрифта элемента.
    • line-height. Устанавливает высоту строки, указывая множитель относительно размера шрифта.
  2. Свойства компоновки текста. Они влияют на интервал и другие особенности компоновки текста, позволяют манипулировать, например, пространством между строками и буквами, а также тем, как текст выравнивается в поле содержимого. Некоторые из таких свойств:
    • text-align. Выравнивание текста. Возможные значения: left — по левому краю (значение по умолчанию), right — по правому краю, center — по центру, justify — по ширине.
    • letter-spacing/word-spacing. Интервал между буквами/словами
    • text-decoration. Оформление текста. Возможные значения: underline — подчёркнутый текст, overline — линия над текстом, line-through — перечёркнутый текст, blink — мигающий текст, none — отмена всех эффектов (значение по умолчанию).
    • text-indent. Отступ первой строки. Возможные значения: length — фиксированное значение в единицах измерения CSS (значение по умолчанию равно 0), % — проценты от ширины строки.
    • text-transform. Трансформация текста. Возможные значения: none — нет трансформации (значение по умолчанию), uppercase — преобразование всех строчных символов в заглавные.
Отступы и размеры

Отступы и размеры в CSS связаны с блочной моделью.

Размеры элементной области задаются с помощью свойств width (ширина элемента) и height (высота элемента). К этим параметрам можно добавлять префиксы min и max для задания минимальной и максимальной ширины и высоты соответственно.

Отступы в CSS бывают двух видов:

  1. Внутренний отступ — это расстояние от воображаемой границы элемента до его содержимого. Величина расстояния задаётся с помощью параметра padding. Такой отступ принадлежит самому элементу и находится внутри него.
  2. Внешний отступ — расстояние между границей текущего элемента веб-страницы и границами соседних элементов, либо родительского элемента. Размер расстояния регулируется свойством margin. Такой отступ находится вне элемента.

По умолчанию внутренние отступы прибавляются к ширине и высоте элемента. Чтобы указать браузеру, что ширина и высота должны включать в себя внутренние отступы и рамки, используется свойство box-sizing.

Медиа-запросы

Медиа-запросы в CSS — это набор правил, которые позволяют адаптировать внешний вид веб-страницы под технические параметры устройства пользователя: ширину и высоту экрана, плотность пикселей, количество поддерживаемых цветов и так далее.

Синтаксис медиа-запросов простой: они начинаются с ключевого слова @media, за которым следует условие и блок стилей, которые применяются, если это условие выполняется.

Некоторые примеры условий:

  • Минимальная ширина экрана. Применяет стили, если ширина экрана больше или равна указанному значению. Пример: @media (min-width: 600px) { body { background-color: lightblue; }}
  • Максимальная ширина экрана. Применяет стили, если ширина экрана меньше или равна указанному значению. Пример: @media (max-width: 600px) { body { background-color: lightgreen; }}
  • Ориентация экрана. Применяет стили в зависимости от ориентации экрана (портретная или ландшафтная). Пример: @media (orientation: landscape) { body { background-color: lightcoral; }}
  • Плотность пикселей. Позволяет изменять стили в зависимости от плотности пикселей на устройстве. Пример: @media (min-resolution: 2dppx) { body { font-size: 18px; }}

Медиа-запросы также позволяют комбинировать несколько условий с помощью логических операторов and, or, not.
